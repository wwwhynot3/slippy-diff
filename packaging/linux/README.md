# Slippy Linux Desktop Bundle

This bundle installs Slippy into your user account without requiring root.

## Install

```bash
./install-linux.sh
```

## Uninstall

```bash
~/.local/bin/slippy-uninstall
```

The raw `slippy` binary can be run directly, but it does not create an application launcher, menu entry, or desktop icon on its own.
